# electron

An Electron wrapper of studio

## Project Setup

### Prerequisites
You should run on dev mode api and studio.

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
